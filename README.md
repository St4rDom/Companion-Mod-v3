# Companion Mod v3 — Forge 1.20.1

## How to get to GitHub
Same process as before — full replace is easiest:
1. Delete all files in your repo (or make a fresh one)
2. Unzip, open companion-mod-v3 folder
3. Press Cmd+Shift+. (Mac) to show hidden files
4. Drag everything into GitHub upload
5. Commit → Actions builds → grab JAR from Artifacts

## What's New in v3

| Feature | Details |
|---------|---------|
| **; key** | Opens Companion Manager (configure in Options → Controls) |
| **Companion Manager** | Scrollable list of all loaded companions. Click any to manage. "Bring All" at top. |
| **Auto-teleport** | If a companion in Follow/Attack/Protect mode is stuck >10 blocks away, it teleports to you (max once/sec) |
| **Bring All button** | In the manager — pulls every loaded companion to your position instantly |
| **Mount control** | WASD now steers the mob while riding. The mob faces your look direction. |
| **Speed sync** | Following companions move at 75% of your current movement speed |
| **Heart particles** | Burst of hearts when you befriend a mob |
| **GUI revamp** | Dark navy theme, accent stripe, segmented health bar, clean sections |

## Controls
- **;** — Open Companion Manager (rebindable in Controls menu)
- **Right-click a mob** — Open that mob's Companion screen
- **WASD while mounted** — Steer the mob
- **ESC** — Close any screen
